## Data safety (declaración)

*Google pide declarar qué datos se recolectan. Marcar:*

- **Personal info — Name**: sí (opcional, usuario lo ingresa)
- **Personal info — Email**: sí (para auth)
- **Personal info — User IDs**: sí (UID de Firebase Auth)
- **App activity — In-app actions**: no (no trackea eventos analíticos)
- **Photos / Videos**: no (la cámara se usa solo para leer barcode, no se guardan imágenes)
- **Device ID**: no
- **Location**: no

Todos los datos se almacenan en Firebase (Google Cloud). No se comparten con terceros salvo los procesadores de pago (Flow.cl cuando el usuario contrata Pro).
